Idea Evaluation UI (Landing Page Mock)

Design a 4:3 product mock UI for a SaaS tool called Creator Content Strategy Copilot.
The UI represents the Idea Evaluation feature used by YouTube creators to evaluate whether a video topic is worth producing.

The UI should clearly illustrate the flow:

Input Idea → AI Analysis → Recommendation

The mock should look like a real product screenshot, suitable for a landing page hero section.

Visual Style

Design Style

minimal modern SaaS UI

clean grid layout

high information density

strong visual hierarchy
