Add Actionable Interaction Module

Modify the existing UI layout to include a high-visibility Actionable Interaction Module placed at the very top of the interface, above the idea input section.

The goal is to visually communicate that the product is not just analysis — it enables creators to take immediate next actions.

New Top Section — Action Hub (Most Prominent Element)

Create a large horizontal card at the top of the UI labeled:

NEXT ACTIONS FOR THIS IDEA

This section should be the most visually prominent element in the UI, even more noticeable than the recommendation card.

Design style:

rounded modern SaaS card

subtle glow or gradient border

slightly elevated shadow

background: white

highlight accents: blue gradient

Include a small AI icon with label:

AI Strategy Actions

Action Buttons (Primary Interaction)

Inside the card display 4 large actionable buttons arranged horizontally.

Buttons should look interactive and clickable.

Each button includes:

action title

short subtitle
